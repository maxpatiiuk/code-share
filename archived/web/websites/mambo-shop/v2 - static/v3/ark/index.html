Про гру:

Ark: Survival Evolved - мультиплеєрна гра в жанрі симулятора виживання. Гра отримала ранній доступ для Microsoft Windows 2 червня 2015 року.
Гравці повинні вижити на острові, наповненому динозаврами, стихійними лихами та іншими потенційно ворожими гравцями. Гра проходить або від третьої, або від першої особи. Доступно переміщення пішки або верхи на динозаврі. Гравці використовують вогнепальну зброю і створюють бази для захисту від інших ворожих гравців і динозаврів. Розробка почалася в жовтні 2014. Команда розробників вивчила фізичні види динозаврів спеціально для гри.
